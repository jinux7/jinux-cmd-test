# jinux-cmd-test
## npm的一个可以全局安装带有cmd下执行命令的package.

### npm package 的发布流程：
> 1)本地开发一个npm的package.
> 2)在此项目的根目录下打开cmd,运行
```
npm adduser
```
> 根据提示填写npm的用户名，密码啥的.
> 3)接着运行
```
npm publish
```
> ok,开发的npm package已经上传到了npm仓库，直接npm install ...就可以喽.
